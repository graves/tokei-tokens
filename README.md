# Tokei Tokens ([時計](https://en.wiktionary.org/wiki/%E6%99%82%E8%A8%88))

Tokei Tokens is a fork of [Tokei](https://github.com/XAMPPRocky/tokei) that extends Tokei to add columns for source code and comment tokens.

Tokei is a program that displays statistics about your code. Tokei will show the number of files, total lines within those files and code, comments, and blanks grouped by language.
